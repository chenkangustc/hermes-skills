## 页面设置

- 页面：A4（21.0 × 29.7 cm）
- 页边距：上 3.7cm / 下 3.5cm / 左 2.8cm / 右 2.6cm
- 行距：28 磅固定值
- 首行缩进：32pt（= 2 字符 @ 16pt）

### Python reportlab 注册示例

```python
from reportlab.pdfbase import pdfmetrics
from reportlab.pdfbase.ttfonts import TTFont

pdfmetrics.registerFont(TTFont('FZXiaoBiaoSong-B05S', '/usr/share/fonts/truetype/gongwen/方正小标宋体.TTF'))
pdfmetrics.registerFont(TTFont('FangSong_GB2312', '/usr/share/fonts/truetype/gongwen/SIMFANG_0.TTF'))
pdfmetrics.registerFont(TTFont('KaiTi_GB2312', '/usr/share/fonts/truetype/gongwen/SIMKAI_0.TTF'))
pdfmetrics.registerFont(TTFont('HeiTi', '/usr/share/fonts/truetype/wqy/wqy-zenhei.ttc'))
# ⚠️ NotoSansCJK-Bold.ttc 是 PostScript 轮廓的 TTC，reportlab 不支持！
pdfmetrics.registerFont(TTFont('TimesNewRoman', '/usr/share/fonts/truetype/msttcorefonts/times.ttf'))
```

> ✅ **方正小标宋简体、仿宋_GB2312、楷体_GB2312 已从用户提供的字体包安装。** 黑体仍使用文泉驿正黑 (wqy-zenhei) 替代。
