# 🔒 专用技能库

> 个人开发的 Hermes 技能存放位置

## 使用方式

1. 在此目录下按分类创建子文件夹
2. 每个技能包含 `SKILL.md`（YAML frontmatter + Markdown 说明）
3. 可选辅助文件：`references/`、`templates/`、`scripts/`
4. 提交到 Git 仓库
