# 中文字体安装流程

如果公文排版需要重新安装字体或在新服务器上设置，按此流程操作。

## 安装依赖

```bash
# 字体解压工具
sudo apt-get install -y unrar

# Microsoft 核心字体（Times New Roman 等）
echo ttf-mscorefonts-installer msttcorefonts/accepted-mscorefonts-eula select true | sudo debconf-set-selections
sudo apt-get install -y ttf-mscorefonts-installer
```

## 安装公文字体包

用户提供的 RAR 包包含 6 个字体文件，全部安装：

```bash
# 解压
mkdir -p /tmp/fonts
cd /tmp/fonts
unrar x <字体包.rar>

# 复制到系统字体目录
sudo mkdir -p /usr/share/fonts/truetype/gongwen
sudo cp *.ttf /usr/share/fonts/truetype/gongwen/

# 刷新字体缓存
sudo fc-cache -f
```

### 字体说明

| 文件名 | 注册字体名 | 用途 |
|--------|-----------|------|
| SIMFANG_0.TTF | FangSong_GB2312 | 正文 仿宋_GB2312 |
| SIMKAI_0.TTF | KaiTi_GB2312 | 二级标题 楷体_GB2312 |
| 方正小标宋体.TTF | FZXiaoBiaoSong-B05S | 公文标题 方正小标宋简体 |
| simfang.ttf | FangSong | 仿宋（备用） |
| simkai.ttf | KaiTi | 楷体（备用） |
| 方正小标宋_GBK.TTF | FZXiaoBiaoSong-B05 | 方正小标宋（备用） |

## 安装黑体（如用户提供）

```bash
# 将 SimHei.ttf 放入
sudo cp SimHei.ttf /usr/share/fonts/truetype/gongwen/
sudo fc-cache -f
```

**当前服务器状态：** 尚未获得 SimHei/黑体，使用文泉驿正黑 (`/usr/share/fonts/truetype/wqy/wqy-zenhei.ttc`) 替代。

## 验证

```bash
# 确认字体可被找到
fc-list | grep -iE "小标|仿宋|楷体|Times|zenhei"
```

## reportlab 注意事项

- Noto Sans CJK 系列是 `.ttc` 文件，包含 PostScript 轮廓 — **reportlab 不支持**
- 遇到 `TTFError: postscript outlines are not supported` 时，改用 `.ttf` 字体或文泉驿系列
- 方正小标宋体.TTF、SIMFANG_0.TTF、SIMKAI_0.TTF 都是单 `.ttf` 文件，无此问题
